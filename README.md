# Cesar Segura Landa - Portafolio profesional

Pagina web personal tipo CV/portfolio para presentar experiencia profesional, proyectos destacados, habilidades tecnicas y datos de contacto.

El sitio esta construido como una pagina estatica lista para publicarse con GitHub Pages.

## Vista general

Este portafolio presenta:

- Perfil profesional como Ingeniero de Software.
- Experiencia en desarrollo de soluciones empresariales.
- Proyectos destacados como portales multiempresa, migracion a Google Cloud y dashboards ejecutivos.
- Stack tecnico con Laravel, Vue.js, JavaScript, Python, PHP, Google Cloud, bases de datos, Git y herramientas de BI.
- Diseno visual dark con estilo espacial, animaciones sutiles y foto de perfil.

## Tecnologias

- HTML5
- CSS3
- JavaScript

- Canvas para animaciones de fondo

## Ejecucion local

Puedes abrir `index.html` directamente en el navegador o levantar un servidor local:

```bash
python -m http.server 4173 --bind 127.0.0.1
```

Despues abre:

```text
http://127.0.0.1:4173/
```

## Publicacion en GitHub Pages

1. Subir los cambios al repositorio.
2. Entrar a `Settings` > `Pages`.
3. Seleccionar la rama principal.
4. Usar la raiz del repositorio como carpeta de publicacion.
5. Guardar y esperar a que GitHub genere la URL publica.
